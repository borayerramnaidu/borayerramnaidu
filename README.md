## 👨‍💻 About Me

Hey there! I'm **Bora Yerram Naidu** — a **DevOps & Cloud Engineer** with **5 years** of experience designing scalable, secure, and highly available cloud-native solutions in **Azure** and **AWS**. I specialize in Kubernetes (AKS, EKS, OpenShift), CI/CD automation, and Infrastructure as Code.

I'm certified across Azure (AZ-104, AZ-400) and AWS (SAA-C03), and I'm passionate about automating everything — from deployments to monitoring.

- 🚀 **Automation addict** — if I do it more than once, I script it
- ☸️ **Kubernetes** is my happy place — AKS, EKS, OpenShift, GitOps with ArgoCD
- 🏗️ **Infrastructure as Code** advocate — Terraform all the way
- 📊 **Observability first** — Prometheus, Grafana & Loki for real-time insights
- ⚡ **Fun fact:** I debug production issues best with good music and coffee ☕🎧
